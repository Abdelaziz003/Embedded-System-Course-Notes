# Embedded Systems Course 🚀

A hands-on Embedded Systems course focused on **real-world applications** and building practical skills from scratch.

---

## 🎯 Course Objective

This course is designed to help you:

* Understand how embedded systems work in real industry environments
* Build complete embedded projects from scratch
* Gain strong practical experience with microcontrollers and drivers

---

## 📚 Course Content

### 🔹 Fundamentals

* Introduction to Embedded Systems
* Microcontroller architecture
* Working with datasheets

### 🔹 Microcontrollers

* ATmega32
* PIC Microcontrollers

### 🔹 Low-Level Drivers

* GPIO
* ADC (Analog to Digital Converter)
* Timers
* USART
* SPI
* I2C

### 🔹 High-Level Drivers

* LED
* LCD
* Keypad
* Sensors
* Motors

### 🔹 Practical Work

* Simulation using Proteus
* Real hardware implementation
* Step-by-step project building

---

## ⚙️ Course Approach

**70% Practical 🛠️ | 30% Theory 📖**

The focus is on learning by doing — not just studying concepts.
